Presentation

Vireo is Landing Page Template with 4 different and responsive layouts with Parallax effects. All Psd-files for free.

Structure of Html Files

All blocks (Promo, Header, Big Features, Small Features, Quotes, Screens, Subscribe and Footer) are located one after the other and use containers with CSS-class Container .

Structure of Css Styles

All css-styles are located in the folder /css. Global styles are located in the file main.css. To reset the default style sheet using a file reset.css. Styles for responsive are located in the file responsive.css.

We got an empty custom.css file which should be used by you to add your own modifications. Since we never touch the custom.css file when making updates you can easily update to later versions without beeing in danger of loosing stylesheet modifications.

Each browser sets its value for different styles of html-tags. To reset the standard tags and ensure cross-browser style sheet is applied reset.css.

Features File global styles main.css:

  • Description style key tag is placed at the top of the file. For example: body, a, h1, h2 и т.д.
  • All css-classes are grouped by adding the name of the class name of the container. For example: css-classes ".header .logo-wrap { ... }", ".header .menu { ... }" refer to the container "head", css-классы ".big-features .title { ... }", ".big-features .box { ... }" - to the container "big-features".

Responsive design is created using CSS3 media queries. The main resolution and page width container container: container:

  • Resolution more than 1300px, container width 1100px.
  • Permission from 980px to 1300px, container width 860px.
  • Permission from 680px to 980px, container width 600px.
  • Resolution of less than 680px, container width 300px.

JavaScript

JavaScript are located in the folder/js.

  • "Stick header to the top" use "fixed_header.js"

  • "Movement" in the promo-area use jQuery-Plugin "FractionSlider" (jQuery parallax Slider Plugin). File location: jquery.fractionslider.js. Settings: jquery.fractionslider.option.js.

  • Modal windows use jQuery-plugin PrettyPhoto. File location: jquery.prettyPhoto.js. Settings: jquery.prettyPhoto.option.js.

  • "Smooth move on page" use jQuery.ScrollTo. File location: jquery.scrollToTop.min.js. Settings: jquery.scrollToTop.option.js.

  • "Manifestations" of objects and promo-zone's effects use animate.js.

  • respond.src.js file provides responsive to IE (less than ver. 9.0).

Settings of animate.js:

1. To control Speed of "manifestations" after the page loads use speedOpacity.

2. To control "manifestations" of scrolling and promo-zone's objects use opacityRate.

3. To control "manifestations" of promo-zone's background use backgroundSpeedRate.

PSD-files

The PSD-files included all layers for Template design (Fool, Comming Soon and News Stream), UI-design and izometric views. PSD-files is fully layered and customizable. Please note that some of the layers are hidden from view. You can easily see the contents of these groups of layers, if you make them "visible".
